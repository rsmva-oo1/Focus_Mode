# 🎯 Focus Master Pro

Focus Master Pro — bu vaqtni to'g'ri boshqarish va ish unumdorligini oshirish uchun mo'ljallangan, zamonaviy va interaktiv **Pomodoro** taymeri. Loyiha yuqori darajadagi UI/UX trendlari (Glassmorphism, Neon Glow) asosida yaratilgan.

---

## ✨ Xususiyatlari (Features)

* **Custom Timer:** Fokus vaqtini o'zingizga moslab o'zgartirish imkoniyati.
* **Professional State Management:** Resume va Reset tugmalari yordamida jarayonni to'liq nazorat qilish.
* **Glassmorphism UI:** Ko'zni charchatmaydigan, premium va zamonaviy interfeys.
* **Micro-interactions:** Dinamik o'zgaruvchan progress-bar va tugmalardagi hover effektlari.
* **Persistence:** Sahifa yangilanganda ham vaqt xotirada saqlanib qoladi (`localStorage`).

---

## 🛠️ Texnologiyalar (Tech Stack)

* **HTML5** — Semantik va toza struktura.
* **CSS3** — Custom properties (`--vars`), CSS animatsiyalar va zamonaviy gradientlar.
* **JavaScript (ES6+)** — Asinxron taymer mantiqi va davomiy holat (state) nazorati.
